### Create a file by vim
```bash
vim README.md
```

insert - i
```bash
i
```
press Esc to escute insert


```bash
:wq
# save changes
```
press ZZ  to exit `git revert hash`
### write content to a new file by echo

#### > means create
```bash
echo "aaa" > test.txt
```

##### >> means append
```bash
echo "aaa" >> test.txt
```
