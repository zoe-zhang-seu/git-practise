# Committing Changes Locally
install vim

```bash
brew install macvim
```
Edit README.md
```bash
vim README.md

i - insert
Esc - excute
:wq - save changes
```

git commit
```bash
git commit -a -m "update readme.md"
```
`-a` means `git add .`
